Compares features in given list of datasets and returns indices of more good and less correlated features

Parameters:
  • data (list of pd.DataFrame) – list of datasets to analyze
  • cross_statistic (cross_statistic) – cross statistical information to change
  • for (str) – for which data make comparison: “true” - for true data, “false” - for false data, “both” - for all data
  • corr_thresholds (float, optional) – which features consider as correlated (if absolute correlation ratio less than given value, features considered as not correlated), default 0.9
Returns:

indices of better features
Return type:

pd.Index

Note

Best feature is a feature which satisfy to some relations in cross statistical information: the lesser mstd/mave, the lesser timpstd for true data (or fimpstd for false data) and the greater timpave for true data (or fimpave for false data)

Statistical information analyzed through datasets (for this algorithm suppose to analyze several datasets with similar features and samples)

mstd/mave:standard deviation of mean values of the feature through datasets divided to averaged mean values of the feature through datasets

computes average for all kind of samples of cross statistical information and compares if with cross statistical information for each feature, then creates boolean map with “True” for good feature and “False” otherwise

Parameters:cross_statistic (pd.DataFrame) – cross statistical information for features
Returns:bool values for features (“True” for good otherwise “False”)
Return type:pd.DataFrame

Note

The feature is better, the lesser mstd/mave, the lesser timpstd for true data (or fimpstd for false data), and the greater timpave for true data (or fimpave for false data) it has (look cross statistical information for meaning mstd/mave, timpstd, fimpstd, timpave and fimpave)

Note

selection_map is nodestimation.learning.selection.make_selection_map() output. Technically speaking, this is usual pd.DataFrame, but its values must be boolean and it must have only two rows: the first one used for true data selection and the second - for false data

  • axis (int, optional) – axis along which to clusterize, default 0
Returns:

Return type:

pd.DataFrame

Note

Clusterization rules

num:the value for each element in the cluster is equated to the ordinal number of this cluster
mean:the value for each element in the cluster is equated to the average of cluster elements
min:the value for each element in the cluster is equated to the minimum of cluster elements
max:the value for each element in the cluster is equated to the maximum of cluster elements
symfar:the value for each element in the cluster is equated to the maximum of cluster elements if the ordinal number of this cluster less than half of clusters number, otherwise to minimum
symclose:the value for each element in the cluster is equated to the minimum of cluster elements if the ordinal number of this cluster less than half of clusters number, otherwise to maximum
