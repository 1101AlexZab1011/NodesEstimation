nodestimation.project.subject module¶

This module provides the main structural unit of NodesEstimation project. This unit represents a patient

class nodestimation.project.subject.Subject(name: str, data: Dict[str, str], nodes: List[nodestimation.Node], directory: str, dataset: pandas.core.frame.DataFrame)[source]¶

Bases: object

Class containing all the necessary information about the patient

Parameters:
  • name (str) – patient`s ID
  • data (dict of str for str) – a dictionary containing paths to patient data files used in calculations. Keys are data types, values are paths to files with this data (or None if they do not exist)
