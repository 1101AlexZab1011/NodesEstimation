nodestimation.pipeline module¶

Module containing pipeline for data transforming and functionality for storing it

nodestimation.pipeline.pipeline(crop_time: Optional[int] = 120, snr: Optional[float] = 0.5, epochs_tmin: Union[int, float, None] = -1, epochs_tmax: Union[int, float, None] = 1, conductivity: Optional[tuple] = (0.3,), se_method: Optional[str] = 'sLORETA', methods: Union[str, List[str], None] = 'plv', rfreq: Optional[int] = 200, nfreq: Optional[int] = 50, lfreq: Optional[int] = 1, hfreq: Optional[int] = 70, freq_bands: Union[tuple, List[tuple], None] = (0.5, 4)) → List[nodestimation.project.subject.Subject][source]¶

Pipeline for brain data transformation

Parameters:
  • methods (list of str or str, optional) – set of metrics to be computed (see list of metrics), default "plv"
  • se_method (str, optional) – MNE solution for inverse computations (see list of MNE solutions), default "sLORETA"
  • conductivity (tuple of float, optional) – suggested tissues conductivity, default (0.3,)
  • epochs_tmin (int or float, optional) – start time (s) before event, default -1
  • epochs_tmax (int or float, optional) – end time (s) after event, default 1
  • rfreq (int, optional) – resampling frequency (Hz), default 200
  • nfreq (int, optional) – frequency (Hz) for notch-filtering, default 50
  • lfreq (int, optional) – frequency (Hz) for low-pass-filtering, default 1
  • hfreq (int, optional) – frequency (Hz) for high-pass-filtering, default 70
  • freq_bands (tuple of float or list of tuple of float) – in what frequency (Hz) diapasons are the calculations performed, default (0.5, 4)
  • crop_time (int, optional) – what time (s) of brain data be read, default 200
  • snr (float, optional) – regularization parameter, default 0.5
Returns:

subjects information, computed according to given parameters
Return type:

list of nodestimation.project.subject.Subject objects
Raises:

ValueError – if freq_bands given in wrong format

Note

metrics that can be calculated: psd, coh, cohy, imcoh, plv, ciplv, ppc, pli, pli2_unbiased, wpli, wpli2_debiased, pearson, envelope

Note

MNE solutions that can be applied: MNE, dSPM, sLORETA, eLORETA
